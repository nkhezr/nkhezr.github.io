Projects

Blockchain-based Model for Sharing Data in Healthcare Applications, Lakehead University
Daily activity recognition patterns provide useful insights to healthcare providers, allowing them to remotely monitor the well-being of the elderly and make appropriate actions. In this project, we proposed a blockchainbased model in Hyperledger for sharing activities of daily living in healthcare applications.

2020
Real-Time Facial Expression Recognition using Convolutional Neural Networks, Lakehead University
In this project, we used Kaggle 2013 challenge about facial expression. Besides benchmark models (AlexNet, VGG16, and VGG19), we created 3 CNN models and trained them from scratch the first model is a simple CNN with 8 layers. The model obtained 61% accuracy on test data. The second model is a deep CNN with 22 Layers, which is accuracy is 82.3%, and the third model with 26 layers obtained 83.6% accuracy on test data.

2019
Fuzzy-based Light Change System for Stop Lights: Based on Pedestrian and Crowd Density, Lakehead University
Extracting features and classification are two-step of detecting objects. In this project, a combination of Histograms of the oriented gradient (HOG) and a multi-class support vector machine (M-SVM) alongside multi-layer perceptron (MLP) is used to gain a better accuracy in pedestrian detection. After detection and extracting number of human pedestrian by HOG and MM-SVM, a fuzzy-based light change system for stop lights is introduced. This system is designed to manage red/green light duration automatically according to the velocity of pedestrian and aims to reduce wasted time.

2019
Introduction to Market Basket Analysis with a Big Data Analytics Approach, Lakehead University
In this project, we proposed a technical study on Market Basket Analysis (MBA) using Big Data Analytics and Machine Learning techniques. The implementation provided by Spark of the FP-Growth Algorithm and Apriori Algorithm.
